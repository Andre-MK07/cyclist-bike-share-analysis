# Cyclistic Bike-Share Analysis

## Overview
This project analyzes Cyclistic bike-share data to uncover behavioral differences between casual riders and annual members. The goal is to translate these insights into actionable marketing strategies and business driven decisions, that will potentially increase membership conversion.

## Business Problem
Cyclistic aims to maximize the number of annual members, as they are more profitable than casual riders. This analysis focuses on understanding how these two groups differ in usage patterns.

## Tools Used
- R (tidyverse, lubridate)
- ggplot2
- Data cleaning & transformation

## Key Insights

- **Commuting Behavior:** Members show strong peaks at 8 AM and 5 PM, indicating routine usage.
- **Leisure Usage:** Casual riders have longer rides (about 60% longer on average) and more flexible patterns.
- **Weekly Trends:** Casual riders are weekend-heavy, while members dominate weekdays.

## Recommendations

- Introduce weekday incentives to build commuting habits among casual riders  
- Target high-frequency casual riders with personalized membership offers  
- Develop flexible or weekend-based membership plans
